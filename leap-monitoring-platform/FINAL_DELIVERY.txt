Common Issues & Solutions:

Issue: Login page keeps showing / flashing
Solution:
  1. Clear browser cache (F12 > Storage > Clear All)
  2. Clear localStorage (F12 > Console > localStorage.clear())
  3. Refresh page
  4. If persists: docker-compose restart leap_dashboard

Issue: Cannot connect to dashboard
Solution:
  1. Check services running: docker-compose ps
  2. Check port 3000: netstat -tuln | grep 3000
  3. View logs: docker-compose logs leap_dashboard
  4. Restart: docker-compose restart leap_dashboard

Issue: Backend API not responding
Solution:
  1. Test health: curl http://localhost:8080/api/v1/health
  2. View logs: docker-compose logs leap_central_collector
  3. Check MongoDB: docker exec leap_mongo mongosh --eval "db.adminCommand('ping')"
  4. Restart all: docker-compose restart

Issue: No data displayed on dashboard
Solution:
  1. Check MongoDB: docker exec leap_mongo mongosh
  2. Check logs in API: curl http://localhost:8080/api/v1/logs
  3. View Central Collector logs: docker-compose logs leap_central_collector
  4. Restart Central Collector: docker-compose restart leap_central_collector

See DEPLOYMENT_GUIDE.md for complete troubleshooting guide.
